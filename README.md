# 短视频剪辑作品集

## 项目简介

这是一个记录短视频剪辑作品和学习过程的仓库。

## 作品展示

### 作品一：医疗科普短视频剪辑

**类型**：医生口播科普内容
**面向群体**：家长
**剪辑要求**：
- 裁剪停顿和口气词
- 适当提速保持节奏
- 字幕排版美观
- 选择合适的BGM
- 制作视频封面
- 保持专业风格（不过于花哨）

**技术栈**：剪映

## 技能展示

- 视频剪辑：剪映（裁剪、字幕、转场、调色、配乐）
- 内容理解：根据受众特点调整剪辑风格
- 封面设计：制作符合内容调性的封面
- 脚本理解：根据文案要求进行剪辑
